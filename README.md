This repository documents a structured 24‑month journey in methodological biostatistics, integrating
foundational mathematics, computational modeling, and public health data science.

- `01_foundations/` — Core mathematics and R programming exercises
- `02_projects/` — Applied public health and statistical modeling projects
- `03_notes/` — Conceptual notes on statistics, epidemiology, and methods
- `04_cheatsheets/` — Quick reference guides and reproducible workflow tips
